# 📊 Excel Sales Dashboard

## 📌 Project Overview

This project involves creating an interactive sales dashboard using Microsoft Excel to analyze business performance and generate insights.

---

## 📈 Dashboard Features

* Sales overview (total revenue, profit, orders)
* Region-wise performance analysis
* Category and sub-category breakdown
* Monthly/Time-based trends
* Interactive filters (slicers)

---

## 🔍 Key Insights

* Identified top-performing regions and categories
* Detected low-performing segments needing improvement
* Observed sales trends over time

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers

---

## 📁 Files Included

* Excel Dashboard File (.xlsx)

---

## 🎯 Purpose

The goal of this project is to demonstrate data analysis and visualization skills using Excel for business decision-making.
